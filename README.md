# Mejiaibot — AI Chatbot (ChatGPT Clone)

**Premium AI Chatbot** built with JavaScript and OpenRouter API.  
Designed for **fast, intelligent responses**, **session persistence**, and **a premium user experience**.

## Features
- Modern **dark mode UI** with smooth animations
- **Markdown & code highlighting** with copy buttons
- **Session management** via `localStorage`
- **Voice recognition** for hands-free interaction
- **Error handling 2.0** — AI responds gracefully under any condition
- Modular architecture with **commands.js** and **utils.js**
- Future-proofed for **Web3/DID integration**

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend API:** OpenRouter API  
- **Utilities:** Clipboard & JSON export helpers  
- **Themes:** Premium enterprise styling
